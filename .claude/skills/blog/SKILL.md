# Notion 기술 블로그 포스트 생성

blog-writer 에이전트에 위임하여 현재 대화 맥락에서 기술 블로그 포스트를 작성하고 Notion에 게시한다.

## 입력

`$ARGUMENTS` = `{유형} {주제}` (예: `설계 헥사고날 아키텍처에서 크로스 컨텍스트 통신`)

- 첫 번째 단어 = 유형 (`설계` | `도입` | `해결`)
- 나머지 = 주제

---

## 워크플로우

### 1. 입력 파싱

`$ARGUMENTS`를 파싱한다:

- **유형**: 첫 번째 단어 (`설계`, `도입`, `해결` 중 하나)
- **주제**: 나머지 텍스트

유효하지 않은 유형이면 안내하고 중단한다:

```
유형은 '설계', '도입', '해결' 중 하나여야 합니다.
- 설계: 설계 의사결정 과정 (문제 해결력 강조)
- 도입: 기술/패턴 도입 사례 (자기 주도성 + 학습 속도 강조)
- 해결: 문제 해결 과정 (문제 해결력 + 자기 주도성 강조)
```

### 2. blog-writer 에이전트 호출

blog-writer 에이전트에 아래 정보를 전달하여 위임한다:

- **유형**: 파싱된 유형
- **주제**: 파싱된 주제
- **맥락**: 현재 대화에서 관련 논의를 추출하여 전달

### 3. 결과 확인

- Notion 페이지 생성 성공 여부 출력
- 생성된 포스트 제목 출력
- Notion 페이지 URL (가능하면) 출력

---

## 사용법

```
/blog 설계 헥사고날 아키텍처에서 크로스 컨텍스트 통신
/blog 도입 AI 워크플로우 고도화
/blog 해결 N+1 쿼리 문제 해결
```
